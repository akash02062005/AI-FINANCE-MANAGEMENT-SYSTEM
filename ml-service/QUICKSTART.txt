AI FINANCE ML MICROSERVICE - QUICK START GUIDE

WHAT YOU HAVE:

================================================================================
STEP 1: INSTALL DEPENDENCIES
================================================================================

$ pip install -r requirements.txt

This installs:
- FastAPI, Uvicorn (web framework)
- scikit-learn, Prophet (ML algorithms)
- pandas, numpy (data processing)
- OpenAI (chatbot integration)
- And 20+ more libraries

Installation time: 5-10 minutes

================================================================================
STEP 2: TRAIN ML MODELS
================================================================================

$ python scripts/train_models.py

This will:
1. Generate 5,000 synthetic financial transactions
2. Train transaction categorizer (TF-IDF + Logistic Regression)
3. Train anomaly detector (Isolation Forest)
4. Train personality analyzer (KMeans clustering)
5. Save all models to trained_models/ directory

Training time: 2-5 minutes
Output: 4 trained model files

================================================================================
STEP 3: CONFIGURE ENVIRONMENT (OPTIONAL)
================================================================================

$ cp .env.example .env

Edit .env for your environment:
- PORT: Change default 8000 to another port if needed
- OPENAI_API_KEY: Add your key for chatbot (optional)
- MONGODB_URI: For production database (optional)

Default configuration works for local development.

================================================================================
STEP 4: RUN THE SERVICE
================================================================================

$ python app.py

Or with auto-reload:

$ uvicorn app:app --reload

Output:
  Uvicorn running on http://0.0.0.0:8000
  Uvicorn running on http://127.0.0.1:8000

Pattern Detection:

Status:

PRODUCTION DEPLOYMENT
================================================================================

   DEBUG=False
   PORT=8000

2. Use production ASGI server:
   gunicorn app:app --workers 4 --worker-class uvicorn.workers.UvicornWorker

3. Set up monitoring:

   - Log errors and requests
   - Track model inference times

4. Configure databases:
   - MongoDB for persistent storage
   - Redis for caching

5. Add authentication:

   - Token-based auth
   - Role-based access control

6. Set up SSL/TLS:
   - HTTPS only
   - Valid certificates
   - Security headers

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "No module named services"
Solution: python -m pip install -r requirements.txt

Problem: "Models not found"
Solution: python scripts/train_models.py
         Creates trained_models/ directory

Problem: "ImportError: No module named openai"
Solution: Chatbot will work with fallback
         Optional: pip install openai

Problem: "Port 8000 already in use"
Solution: uvicorn app:app --port 8001

Problem: "Connection refused (MongoDB)"
Solution: MongoDB is optional
         Default configuration works without it
         For data persistence, install MongoDB
